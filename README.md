# Bonemerge
A blender addon to snap TF2 cosmetic to a player rig easily. 

Made by hisanimation and Herwork

## How to install

1: Download the file
2: Open blender preferences
3: Go to the addon tab
4: Click install
5: Locate to the "bonemerge.py"
6: And install :D
